# 安装

本项目已内置全部依赖，执行以下命令安装：

```bash
pnpm install
```

## 依赖说明

| 依赖 | 作用 |
| --- | --- |
| @rspress/core | 文档框架核心 |
| @rspress/plugin-preview | 组件 iframe 预览 |
| @rspress/plugin-playground | 在线代码编辑 |
| @rsbuild/plugin-vue | 编译 .vue 单文件组件 |
| @rsbuild/plugin-vue-jsx | 编译 .tsx / .jsx（Vue JSX） |
| @rsbuild/plugin-babel | 为 vue-jsx 提供 Babel 能力 |
| @rsbuild/plugin-less | 编译 Less 样式 |
| element-plus | UI 组件库 |
| vue | 组件运行时 |

## 可用脚本

| 命令 | 说明 |
| --- | --- |
| `pnpm dev` | 启动开发服务器 |
| `pnpm build` | 构建生产产物到 `doc_build/` |
| `pnpm preview` | 本地预览生产产物 |
