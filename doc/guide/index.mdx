# 介绍

`vue3-component-docs` 是一个基于 [Rspress](https://rspress.dev) 搭建的组件库文档站点，用于预览与渲染基于 **Vue3 + TSX + Less + Element Plus** 的组件。

## 特性

- 基于 Rspress 的静态文档生成，构建快、体验好
- 使用 `@rspress/plugin-preview` 在 iframe 沙箱中预览 Vue3 组件
- 同时支持 `.vue` 单文件组件与 `.tsx` 两种写法
- 内置 Less 预处理器，支持变量、嵌套与 `@import`
- 预览环境全局注册 Element Plus，开箱即用

## 技术栈

| 能力 | 方案 |
| --- | --- |
| 文档框架 | Rspress v2 |
| 组件渲染 | Vue 3 |
| TSX 支持 | @rsbuild/plugin-vue-jsx + @vue/babel-plugin-jsx |
| 样式预处理 | @rsbuild/plugin-less |
| UI 组件库 | Element Plus |
| 组件预览 | @rspress/plugin-preview（iframe-follow 模式） |

## 目录结构

```bash
vue3-component-docs/
├── rspress.config.ts   # 站点与预览插件配置
├── doc/                # 文档根目录
│   ├── _nav.json       # 顶部导航
│   ├── index.mdx       # 首页
│   ├── guide/          # 指南文档
│   └── components/     # 组件文档与示例
└── package.json
```
