# 编写组件示例

本项目通过 `@rspress/plugin-preview` 在 iframe 沙箱中渲染组件，支持以下文件作为示例：

- `.vue` 单文件组件
- `.tsx` / `.jsx`（Vue JSX）

## 预览模式

所有示例使用 `preview="iframe-follow"` 模式：组件渲染在跟随代码块的 iframe 中，与文档站点样式完全隔离。该模式支持自定义入口（`customEntry`），是 Vue3 组件预览的推荐模式。

## 内联示例

直接在代码块中编写组件代码，默认导出组件即可：

```tsx preview="iframe-follow"
import { defineComponent } from 'vue';
import { ElTag } from 'element-plus';

export default defineComponent({
  setup() {
    return () => (
      <div style={{ padding: '16px' }}>
        <ElTag type="success">内联 TSX 示例</ElTag>
      </div>
    );
  },
});
```

## 外部文件示例

将组件代码放在单独文件中，通过 `file` 属性引用（示例代码以 `_` 开头会被排除出路由）：

````mdx
```vue file="./_demo-button.vue" preview="iframe-follow"

```
````

## 在线编辑（Playground）

使用 `playground` 语法可在浏览器中**实时编辑**代码并查看效果。本项目的 playground 已通过自定义渲染（`playground/Playground.tsx`）支持 Vue3：编辑左侧代码，右侧预览实时更新。

```tsx playground
import { defineComponent, ref, h } from 'vue';

export default defineComponent({
  setup() {
    const count = ref(0);
    return () =>
      h('div', { style: 'padding: 16px; font-family: system-ui, sans-serif' }, [
        h('p', { style: 'font-size: 16px; margin: 0 0 12px' }, `当前计数：${count.value}`),
        h(
          'button',
          {
            style:
              'padding: 6px 16px; background: #409eff; color: #fff; border: none; border-radius: 4px; cursor: pointer',
            onClick: () => (count.value += 1),
          },
          '点击 +1',
        ),
      ]);
  },
});
```

:::warning playground 的限制
playground 在浏览器端用 `@babel/standalone` 编译，**不包含** `@vue/babel-plugin-jsx`，因此：
- 支持 `h()` 渲染函数、`template` 字符串写法；
- **不支持** JSX/TSX 语法（`<div>`）；
- 依赖需通过 `include`（如 `vue`）预打包，无法引用本地文件。

需要 JSX 或本地文件预览时，请改用上方的 `preview="iframe-follow"`。
:::

## .vue 与 .tsx 的差异

| 写法 | 组件引用 | 说明 |
| --- | --- | --- |
| `.vue` | 模板中使用 `<el-button>`（kebab-case） | 依赖全局注册的 Element Plus |
| `.tsx` | JSX 中使用 `<ElButton>`（PascalCase）并显式 import | JSX 中仅 PascalCase 会解析为组件 |

:::warning
在 TSX 中，`<el-button>` 会被当作原生 HTML 元素，不会解析为组件。请使用 `<ElButton>` 并显式导入。
:::
