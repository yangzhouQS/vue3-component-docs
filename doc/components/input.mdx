## 属性

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| model-value | 绑定值 | `string` | `''` |
| placeholder | 占位文本 | `string` | `''` |
| disabled | 是否禁用 | `boolean` | `false` |
| clearable | 是否可清空 | `boolean` | `false` |
