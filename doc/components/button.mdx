# Button 按钮

常用的操作按钮，用于触发即时操作。

## 基础用法

使用 `.vue` 单文件组件 + Less 编写，模板中通过 `<el-button>` 使用 Element Plus。

## 在线编辑（可修改代码）

下方为 `playground` 示例，**可直接在右侧编辑器修改代码，左侧实时预览效果**（使用 `h()` 渲染函数写法）。

```tsx playground
import { defineComponent, ref, h } from 'vue';

export default defineComponent({
  setup() {
    const count = ref(0);
    return () =>
      h('div', { style: 'padding: 16px; display: flex; gap: 12px; align-items: center' }, [
        h(
          'button',
          {
            style:
              'padding: 8px 18px; background: #409eff; color: #fff; border: none; border-radius: 4px; cursor: pointer; font-size: 14px',
            onClick: () => (count.value += 1),
          },
          `点击 ${count.value}`,
        ),
        h('span', { style: 'color: #606266' }, '可在右侧编辑器修改代码'),
      ]);
  },
});
```

### preview 与 playground 的区别
- `preview="iframe-follow"`：完整渲染（支持 TSX/JSX、Less、Element Plus），代码为**只读展示**。
- `playground`：代码**可在线编辑**实时预览；支持原始 SFC（`<template>` + `<script setup>` + `<style>`）写法。


## 在线编辑 - 原始 SFC 写法

支持完整 Vue 单文件组件语法（`<template>` + `<script setup lang="ts">` + `<style scoped>`），可直接使用 Element Plus 组件与图标，右侧编辑器修改后实时预览：

```tsx playground
<template>
  <div class="button-example">
    <div class="button-row">
      <el-button>Default</el-button>
      <el-button type="primary">Primary</el-button>
      <el-button type="success">Success</el-button>
      <el-button type="info">Info</el-button>
      <el-button type="warning">Warning</el-button>
      <el-button type="danger">Danger</el-button>
    </div>

    <div class="button-row">
      <el-button :icon="Search" circle />
      <el-button type="primary" :icon="Edit" circle />
      <el-button type="success" :icon="Check" circle />
      <el-button type="warning" :icon="Star" circle />
      <el-button type="danger" :icon="Delete" circle />
    </div>
  </div>
</template>

<script lang="ts" setup>
import { Check, Delete, Edit, Search, Star } from '@element-plus/icons-vue'
</script>

<style scoped>
.button-example {
  display: flex;
  flex-direction: column;
  gap: 1rem;
  padding: 16px;
}

.button-row {
  display: flex;
  flex-wrap: wrap;
  gap: 1rem;
  align-items: center;
}
</style>

```

## 外部文件在线编辑

通过 `file` 引入外部 `.vue` 文件，同样支持在线编辑。注意：

- 代码块语言须用 `tsx`（playground 仅识别 `jsx`/`tsx`），`file` 指向 `.vue` 文件即可；
- 样式需用普通 `<style>`（**不能用 `lang="less"`/`scss`**，浏览器端 playground 无 CSS 预处理器）。

```tsx file="./_demo-button-play.vue" playground

```

## 属性

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| type | 按钮类型 | `'primary' \| 'success' \| 'warning' \| 'danger' \| 'default'` | `'default'` |
| disabled | 是否禁用 | `boolean` | `false` |
| loading | 是否加载中 | `boolean` | `false` |
